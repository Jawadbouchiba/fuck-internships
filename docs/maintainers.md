# Maintainers Guide

## Responsibilities

- Keep scope focused on student outcomes.
- Review and merge quality contributions.
- Enforce code of conduct.
- Keep onboarding path fast and clear.

## Triage routine

- Label new issues within 3 business days.
- Mark beginner-safe tasks as `good first issue`.
- Close unclear or stale issues with explanation.

## Review standards

- Require issue linkage for non-trivial PRs.
- Ask for evidence fields in PR template.
- Prefer small, iterative contributions.

## Contact

Maintainer contact methods should be listed in repository Discussions or a pinned issue.
