# Computer Science Module

This module focuses on core CS fundamentals through practical open-source tasks.

## Suggested mini-projects

- Data Structures & Algorithms notes + benchmark exercises
- Graph/Tree problem walkthrough repository docs
- Complexity analysis examples with test cases

## Starter contribution ideas

- Write or improve one algorithm explanation doc
- Add unit tests for an existing algorithm implementation
- Create reproducible benchmark script for one algorithm variant

## What to submit

- linked issue + PR,
- explanation of approach and complexity,
- test/benchmark evidence,
- reflection in showcase template.

## Skill signals for recruiters

- problem-solving,
- analytical reasoning,
- correctness and performance thinking.
