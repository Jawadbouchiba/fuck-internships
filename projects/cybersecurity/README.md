# Cybersecurity Module

This module focuses on defensive security, secure coding, and practical security operations.

## Suggested mini-projects

- Secure coding checklist and vulnerable-pattern examples
- Threat-model template improvements for open-source projects
- Dependency/security scanning workflow hardening

## Starter contribution ideas

- Add security-focused docs for one common vulnerability category
- Improve CI checks with security scan integration notes
- Review and report one risky pattern with remediation guidance

## What to submit

- linked issue + PR,
- risk description and severity rationale,
- mitigation steps and verification evidence,
- reflection in showcase template.

## Skill signals for recruiters

- security awareness,
- risk-based reasoning,
- secure development practices,
- clear communication of findings.
