# Software Engineering Module

This module focuses on shipping maintainable software with team workflows.

## Suggested mini-projects

- API feature implementation with tests
- CI workflow improvements for docs/code checks
- Refactor task improving readability and reliability

## Starter contribution ideas

- Implement one scoped bug fix with regression test
- Improve one CI workflow or repository automation
- Add onboarding tooling or developer docs that reduce setup friction

## What to submit

- linked issue + PR,
- architecture/trade-off notes,
- tests and validation output,
- reflection in showcase template.

## Skill signals for recruiters

- clean implementation,
- testing discipline,
- collaboration in PR workflow,
- maintainability mindset.
